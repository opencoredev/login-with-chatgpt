# Login with ChatGPT docs

Fumadocs/Next.js site for the SDK documentation.

```bash
bun install
bun run dev
```

Default Next URL: http://localhost:3000. When running beside the demo, use:

```bash
PORT=3001 bun run dev
```

Useful commands:

```bash
bun run types:check
bun run build
bun run lint
```

Content lives in `content/docs`. The homepage lives in `src/app/(home)/page.tsx`.
